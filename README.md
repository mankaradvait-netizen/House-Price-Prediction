# House Price Prediction

## Project Overview
This project predicts house prices using Machine Learning techniques. The dataset contains information such as area, bedrooms, bathrooms, parking spaces, furnishing status, and amenities.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Seaborn

## Models Used
1. Linear Regression
2. Random Forest Regressor

## Results

| Model | R² Score |
|---------|---------|
| Linear Regression | 0.714 |
| Random Forest | 0.683 |

Linear Regression achieved the best performance with an R² score of 0.714.

## Key Insights
- Area is the strongest predictor of house price.
- Bathrooms and parking significantly impact price.
- Air conditioning and preferred area increase property value.
- Linear Regression outperformed Random Forest on this dataset.
